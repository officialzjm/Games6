# Code of Conduct

## Our Pledge

We want this project to be welcoming for people learning, testing, fixing bugs, and improving retro web emulation. Be respectful, helpful, and patient with contributors of all experience levels.

## Expected Behavior

- Use welcoming and constructive language.
- Respect different skill levels, backgrounds, and viewpoints.
- Give feedback about the code or idea, not the person.
- Credit upstream projects and contributors when discussing emulator work.
- Keep legal and licensing discussions calm, factual, and source-based.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Sharing copyrighted ROMs, BIOS files, or links to pirated content.
- Removing attribution from upstream emulator projects.
- Claiming third-party emulator work as your own.
- Spamming issues, pull requests, or discussions.

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or block users when behavior harms the project or community.

If you need to report a problem, open a GitHub issue with the relevant context. If the issue involves sensitive personal information, keep details minimal and ask for a private contact path.

## Scope

This code of conduct applies to this repository, issues, pull requests, discussions, and any public project spaces connected to this repository.
